\section{Open Source Release}
\label{sec:release}

The public release includes:
\begin{itemize}
  \item GitHub: \url{https://github.com/gregyoung14/openmarket} (tag
        \OpenMarketReleaseTag{})
  \item Hugging Face dataset: \texttt{gregyoung14/openmarket-btc-polymarket}
        (\texttt{v0.4.3-unified}, \texttt{v0.2-full}, \texttt{v0.1-sample})
  \item Hugging Face models: \texttt{gregyoung14/openmarket-models}
        (\texttt{v0.2.1/} walk-forward logistic on unified step3;
        \texttt{v0.1/} historical)
  \item mdBook documentation and Rust API documentation
  \item Apache License 2.0 (code) and documented HF dataset license
  \item Citation guide: \path{CITATION.md} (BibTeX for dataset, code, models)
  \item Docker reproducibility, benchmark tables, and contribution guide
  \item Jupyter quickstart: \path{notebooks/quickstart.ipynb} (HF sample split)
  \item GitHub issues labeled by contribution area (assets, exchanges, models)
\end{itemize}
