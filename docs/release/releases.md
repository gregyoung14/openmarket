# Release Process

Each public release should have:

- source tag
- changelog
- dataset version
- model version
- benchmark table
- known limitations

Example:

```text
v0.1.0
Source: github.com/gregyoung14/openmarket/releases/tag/v0.1.0
Dataset: huggingface.co/datasets/gregyoung14/openmarket-btc-polymarket/v0.1
Models: huggingface.co/gregyoung14/openmarket-models/v0.1
Benchmarks: docs/release/benchmarks-v0.1.md
```
